# Colosseum 콜로세움
* 논쟁거리를 모아 토론하는 포럼 커뮤니티
* 게시글을 작성해 안건을 추가할 수 있다.
* 로그인 없이 논쟁거리를 확인할 수 있으나 댓글 및 게시글 작성등은 로그인 이후에 가능하다.

# 구현 목록
## 도메인
- [X] 회원
- [X] 회원 저장소

## 메인 페이지 (추후 구현)
- [ ] 현재 핫한 토론 목록을 보여준다.
- [ ] 최근 추가된 토론 목록을 보여준다.

## 회원가입
- [X] 입력된 데이터를 검증하여 회원을 추가한다.
  - [X] 아이디 중복 검증
  - [X] 비밀번호 일치 검증
  - [X] 필수 입력 사항을 기입해야 회원이 추가된다.
  - [X] 선택 입력 사항 데이터를 추가한다 (추후 구현)
- [X] 로그인 페이지 이동 버튼 구현
- [X] DTO 사용 (추후 구현)
- [X] 에러 처리 위치 수정
- [X] 에러 메시지 수정
- [X] 성별 선택 라디오로 구현


## 로그인
- [X] 입력된 데이터를 검증한다.
  - [X] 이메일 및 비밀번호 입력 확인
- [X] 일치하는 회원이 있는지 확인한다.
  - [X] 있는 경우 메인 페이지로 이동
  - [X] 없는 경우 안내메시지와 함께 비밀번호 입력칸 초기화
- [X] 회원가입 페이지 이동 버튼 구현
- [ ] 비밀번호 찾기 페이지 이동 버튼 구현 (추후)
- [ ] 로그인 이후 이전 페이지 이동 (추후)
